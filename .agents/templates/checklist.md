# Finish Checklist

Use this checklist before closing or handing off a task.

- The primary owner matches the main edited files.
- Any cross-boundary edits are minimal and directly required.
- All required checks for the active subagent were run, or any skipped checks are called out explicitly.
- Docs were updated only if user-facing behavior, workflow, or ownership guidance changed.
- No unrelated files were edited.
- Any handoff uses `.agents/templates/handoff.md` exactly.
